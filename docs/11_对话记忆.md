# 对话记忆（Conversation Memory）

## 问题背景

多轮对话有三个核心挑战：

1. **指代消解** — 用户说"她"、"那个"，LLM 需要知道指什么
2. **上下文窗口** — 对话越来越长，LLM 装不下
3. **信息压缩** — 需要选择性保留关键信息

```
第 1 轮：用户问"沈令月是什么身份？" → 回答"她是公主"
第 2 轮：用户问"她为什么要选谢初？" → LLM 需要知道"她"= 沈令月
第 10 轮：用户问"她之前提到的那个人是谁？" → 需要回溯早期对话
```

## 三层记忆架构

```
长期记忆（实体追踪）
  └── 沈令月 = 公主嫡女
  └── 谢初 = 昭武将军
  └── 顾大人 = 青梅竹马
        ↓ 贯穿整个对话，不会丢失

中期记忆（摘要压缩）
  └── "沈令月选谢初为驸马，谢初在长林盛宴中拔得头筹..."
        ↓ 早期对话压缩为 50-100 字摘要

短期记忆（滑动窗口）
  └── 最近 3 轮对话原文
        ↓ 保留最新的上下文细节
```

## 实现

```python
class ConversationMemory:
    def __init__(self, window_size=3):
        self.window_size = window_size    # 滑动窗口大小
        self.history = []                 # 完整对话历史
        self.summary = ""                 # 早期对话摘要
        self.entities = OrderedDict()     # 实体追踪

    def add(self, role, content):
        """添加一轮对话"""
        self.history.append({"role": role, "content": content})
        if role == "user":
            self._extract_entities(content)

    def resolve(self, question):
        """指代消解：把'她'、'他'替换为具体实体"""
        persons = [e for e, d in self.entities.items() if "人物" in d]
        resolved = question
        for p in ["她", "他", "那个"]:
            if p in resolved and persons:
                resolved = resolved.replace(p, persons[0], 1)
        return resolved

    def _summarize_old(self):
        """把超出窗口的早期对话压缩成摘要"""
        if len(self.history) <= self.window_size * 2:
            return
        old = self.history[:-self.window_size * 2]
        hist_text = "\n".join(f"{'用户' if h['role']=='user' else '助手'}: {h['content']}" for h in old)
        self.summary = call_llm([
            {"role": "system", "content": "总结以下对话的关键信息，50字以内。"},
            {"role": "user", "content": hist_text}
        ], max_tokens=80)
        self.history = self.history[-self.window_size * 2:]

    def get_context(self):
        """获取构建 prompt 所需的上下文"""
        parts = []
        if self.entities:
            parts.append(f"[已知人物] {'、'.join(f'{n}（{d}）' for n,d in self.entities.items())}")
        if self.summary:
            parts.append(f"[对话摘要] {self.summary}")
        recent = self.history[-self.window_size * 2:]
        if recent:
            parts.append(f"[最近对话]\n" + "\n".join(f"{'用户' if h['role']=='user' else '助手'}: {h['content']}" for h in recent))
        return "\n\n".join(parts)
```

## 效果

```
[第 1 轮] 沈令月是什么身份？
  → 已知实体: []

[第 2 轮] 她为什么要选谢初当驸马？
  → [指代消解] '她' → '沈令月'

[第 4 轮] 沈令月和顾大人的关系是什么？
  → [记忆压缩] 早期对话已压缩为摘要: 沈令月是公主嫡女...
  → 已知实体: ['沈令月和顾大人']

[第 5 轮] 她最后的选择是什么？
  → [指代消解] '她' → '沈令月和顾大人'
  → 对话摘要: 沈令月选谢初为驸马...
```

## 适用场景

- 多轮对话系统（客服、助手）
- 需要跨轮次推理的问题
- 对话历史较长，需要压缩

## 实际生产中的优化

- 实体提取用 LLM 做（比正则更准）
- 摘要用专门的 Summary 模型（比 LLM 快且便宜）
- 滑动窗口大小根据 context window 动态调整
